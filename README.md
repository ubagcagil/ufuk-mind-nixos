# Ufuk's NixOS Mind

This is my personal NixOS flake – a "brain" that manages multiple machines:

- **ufuk-desktop** → main base / workstation + future homelab server
- **ufuk-laptop** → mobile scout client (WireGuard VPN to desktop)

## Philosophy

- Minimal, zen, no bloat
- Self-hosting and techno-anarchist leanings (degoogle, demicrosoft, de-state)
- Single source of truth: `/etc/nixos` (flake + git)
- "Weighted life": body, mind and machines as one training ground

## Layout

- `flake.nix`  
  → Defines all NixOS profiles: `ufuk-desktop`, `ufuk-laptop` (and future hosts).

- `modules/common.nix`  
  → Shared configuration for all hosts (locale, GNOME, PipeWire, firewall, SSH, sudo, basic packages, etc.).

- `hosts/ufuk-desktop.nix`  
  → Desktop-specific config (NVIDIA RTX 4090, WireGuard server, NAT, "never sleep" power settings).

- `hosts/ufuk-laptop.nix`  
  → Laptop-specific config (zram, power profiles, WireGuard client).

- `hardware-*.nix`  
  → Per-machine hardware configs generated by `nixos-generate-config`, slightly adapted.

## How to use (for my machines)

> This repo contains no secrets. Private keys live in `/etc/wireguard/*` on each machine and are **not** checked into git.

Typical workflow:

1. Edit config under `/etc/nixos` on ufuk-desktop.
